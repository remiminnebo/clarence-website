// MULTIPLAYER · LIVE

The cockpit goes
multiplayer.

Clarence is the only AI coding tool with real-time collaboration built in. Watch a teammate's agent work live. Send it a task. Merge your cockpits into one. All over an encrypted channel — no screen sharing required.

OBSERVE

Watch any agent,
live.

Request to observe a friend's terminal. Their PTY output streams to a live read-only xterm pane in your cockpit — every keystroke, every token. The moment you want to know how they're tackling a problem, just ask to observe.

  • Real-time terminal stream over Ably
  • Friend accepts or declines the request
  • LIVE badge · read-only · zero lag

HANDOFF

Pass a task to
any agent.

Write a prompt and send it directly into a friend's agent session. Their agent picks it up as the next instruction — no copy-paste, no Slack messages, no context lost in translation. One click hands off the work.

  • Prompt delivered straight to their PTY
  • Recipient sees a toast to accept or decline
  • Full context preserved end-to-end

⊞

Tiled panes

Split your screen into 1, 2, 4, 6, 8, or 12 panes. Each runs a fully independent PTY — resize, close, or zoom any pane with a keystroke.

⟳

Any agent

Claude Code, opencode, Codex, or any CLI tool that speaks a terminal. Mix and match agents across panes in the same workspace.

◈

Workspace isolation

Group panes into named workspaces — each with its own repo, branch, and layout. Switch contexts instantly; state persists across restarts.

⊙

GitHub PR review

See your open PRs in the sidebar. Click one to inspect CI checks and review comments, then spawn an agent to review it — or review all at once.

●

Friends & presence

See which of your GitHub mutual follows are running Clarence right now and how many agents they have active. Ambient awareness, no chat required.

◎

Accent colors

Each workspace gets its own color. Pane borders, sidebar indicators, and agent groups all follow it — so your eyes never lose context.
